# 🎬 TMDB Lite

TMDB Lite is a modern movie discovery and recommendation platform built with React, TypeScript, and Vite.  
It integrates with the TMDB API to provide real-time movie data, search functionality, and dynamic browsing experience.

---

## 🚀 Features

- 🔍 Search for movies in real-time
- 🎥 View trending and popular movies
- 📄 Detailed movie information pages
- ⭐ Movie ratings and overview display
- 📱 Fully responsive design
- ⚡ Fast performance powered by Vite
- 🎨 Clean UI built with Tailwind CSS

---

## 🛠 Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- TMDB API

---

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/Afolabiyinka/TMDB-Lite--.git
cd TMDB-Lite--
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start development server

```bash
npm run dev
```

App runs at:

```
http://localhost:5173
```
